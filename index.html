把列表（或任何可迭代的东西）lst 里面的每一个元素，依次取出来，暂时命名为 x，然后对这个 x 做你想做的事情（这里是打印它）。

for in range

这是 Python 写”计数循环”最常用的方式

常用写法

想实现的效果 推荐写法 说明 / 对比 C/C++
循环 n 次（从 0 开始） for i in range(n): 相当于 for(int i=0; i<n; i++)
从 1 到 n（包含 n） for i in range(1, n+1): 很常见，尤其处理 1-based 索引
倒序 n → 1 for i in range(n, 0, -1): 相当于 for(int i=n; i>=1; i--)
每隔 k 个数 for i in range(0, n, k): 步长 k
同时遍历索引和元素 for i, v in enumerate(lst): 相当于带下标的循环（很常用）
同时遍历两个列表 for a, b in zip(list1, list2): 相当于最短的那个列表结束就停止
遍历字典 for k in d: 或 for k,v in d.items(): 默认只遍历 key；用 .items() 拿键值对

if / elif / else

基本规则

  • 条件后面一定要写冒号 :
  • 条件不需要写小括号（写也合法，但几乎没人写）
  • 代码块靠缩进（通常 4 个空格）来界定范围
  • 没有 {}，所以缩进错误会直接报错（IndentationError）

比较运算符

运算符 含义 示例 C/C++ 是否相同
== 相等 x == 5 相同
!= 不相等 x != "hello" 相同
> / >= 大于 / 大于等于 age >= 18 相同
< / <= 小于 / 小于等于 score < 60 相同
is 同一个对象（身份） x is None 无直接对应
is not 不是同一个对象 x is not y 无直接对应
in 成员包含 "a" in "apple" 无直接对应
not in 不包含 5 not in [1,2,3] 无直接对应

常见写法 & 容易踩的坑

场景 推荐写法（Python 风格） 不推荐 / 容易错的写法 说明
判断是否为空 if not name: if name == "": 更 Pythonic，适用于 str/list/tuple/dict 等
判断 None if result is None: if result == None: 用 is 更安全（虽然很多时候 == 也行）
多条件优先级 if a>0 and b>0 or c>0: — and优先级高于or，和 C 一样
链式比较 if 0<x<100: if x>0 and x<100: 链式更简洁可读
否定复杂条件 if not ( age>=18 and has_ticket): — not 作用于整个括号
条件中包含 in / not in if "error" in message.lower(): — 非常常用，尤其处理字符串和列表

使用 PyInstaller

最简单尝试（目录版，调试用）

选项 说明
--windowed macOS GUI 程序专用（等同于 --noconsole）

生成结果：dist/我的工具.app（可直接双击运行）

推荐方案（单文件 + 干净）

关键选项说明：

选项 说明
--onefile 生成单个可执行文件（macOS 会生成 .app 包结构）
--add-data 显式加入 customtkinter 资源（字体、主题 JSON），防止界面崩坏
--hidden-import 强制包含 tkinter 和 customtkinter（分析器可能漏掉）

如果有额外资源文件（图片、json、字体等）

macOS 常见问题

问题 现象 解决方法
双击 .app 闪退 / 无反应 Finder 启动崩溃，Terminal 运行正常 添加 --windowed --osx-bundle-identifier "com.yourname.mytool"，检查 Python 是否为 Homebrew 安装
界面丑 / 无圆角 / 主题丢失 打包后变回原生 tk 丑陋风格 加 --add-data 把 customtkinter 整个目录加进去
（ModuleNotFoundError） 运行时报 tkinter 或 customtkinter 找不到 加 --hidden-import tkinter --hidden-import customtkinter
图标不生效 .icns 不显示 图标文件必须是 .icns 格式，用 Preview.app 转换
体积过大 包含了很多无用依赖 用 --exclude-module 排除不用的模块（如 matplotlib、numpy 等）
Gatekeeper 阻止运行 “来自身份不明的开发者” 自签名：codesign --force --deep --sign - dist/MyTool.app
