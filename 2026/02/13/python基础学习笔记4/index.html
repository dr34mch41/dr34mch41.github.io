for in

把列表（或任何可迭代的东西）lst 里面的每一个元素，依次取出来，暂时命名为 x，然后对这个 x 做你想做的事情（这里是打印它）。
样例：

for in range

这是 Python 写”计数循环”最常用的方式

常用写法

想实现的效果 推荐写法 说明 / 对比 C/C++
循环 n 次（从 0 开始） for i in range(n): 相当于 for(int i=0; i<n; i++)
从 1 到 n（包含 n） for i in range(1, n+1): 很常见，尤其处理 1-based 索引
倒序 n → 1 for i in range(n, 0, -1): 相当于 for(int i=n; i>=1; i--)
每隔 k 个数 for i in range(0, n, k): 步长 k
同时遍历索引和元素 for i, v in enumerate(lst): 相当于带下标的循环（很常用）
同时遍历两个列表 for a, b in zip(list1, list2): 相当于最短的那个列表结束就停止
遍历字典 for k in d: 或 for k,v in d.items(): 默认只遍历 key；用 .items() 拿键值对
