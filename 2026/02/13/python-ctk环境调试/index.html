推荐方案（单文件 + 干净）

关键选项说明：

选项 说明
--onefile 生成单个可执行文件（macOS 会生成 .app 包结构）
--add-data 显式加入 customtkinter 资源（字体、主题 JSON），防止界面崩坏
--hidden-import 强制包含 tkinter 和 customtkinter（分析器可能漏掉）

如果有额外资源文件（图片、json、字体等）

macOS 常见问题

问题 现象 解决方法
双击 .app 闪退 / 无反应 Finder 启动崩溃，Terminal 运行正常 添加 --windowed --osx-bundle-identifier "com.yourname.mytool"，检查 Python 是否为 Homebrew 安装
界面丑 / 无圆角 / 主题丢失 打包后变回原生 tk 丑陋风格 加 --add-data 把 customtkinter 整个目录加进去
（ModuleNotFoundError） 运行时报 tkinter 或 customtkinter 找不到 加 --hidden-import tkinter --hidden-import customtkinter
图标不生效 .icns 不显示 图标文件必须是 .icns 格式，用 Preview.app 转换
体积过大 包含了很多无用依赖 用 --exclude-module 排除不用的模块（如 matplotlib、numpy 等）
Gatekeeper 阻止运行 “来自身份不明的开发者” 自签名：codesign --force --deep --sign - dist/MyTool.app
